<h1 align="center">AcademiaOS</h1>

Добро пожаловать в **AcademiaOS** — ваше универсальное решение для поиска академической информации и анализа данных! Мы построили эту платформу на базе мощных больших языковых моделей с набором функций, предназначенных для максимальной поддержки исследователей.

Для более глубокого понимания лежащих в основе технологий и концепций вы можете ознакомиться с нашей статьёй: [AcademiaOS: Автоматизация разработки теорий обоснованной теории в качественных исследованиях с помощью больших языковых моделей](https://arxiv.org/abs/2403.08844).

## 🌟 Возможности

* **Поиск академических статей**: Основываясь на корпусе SemanticScholar и эмбеддингах OpenAI, AcademiaOS находит и ранжирует релевантные статьи по вашим запросам.
* **Загрузка PDF-файлов**: Если у вас есть отобранные статьи или другие качественные документы (например, стенограммы интервью) в формате PDF, вы можете загрузить их для последующей обработки. PDF с текстом обрабатываются в браузере, а отсканированные PDF распознаются через Adobe PDF Extract API.
* **Массовое извлечение информации**: Структурированно извлекайте информацию (например, позицию статьи по отношению к вашей гипотезе или количество участников исследования) из статей в масштабе.
* **Автоматизированный обзор литературы**: Навигация через чистый и интуитивно понятный интерфейс.
* **Кодирование качественной литературы**: Позвольте ИИ кодировать ваши интервью, публикации в социальных сетях или другую качественную литературу.
* **Автоматическое построение теорий**: Получите теоретическую модель, объясняющую ваши качественные данные, всего за несколько шагов.

## 🔧 Начало работы

Технологический стек:
- ReactJS
- AntDesign (библиотека компонентов)
- LangChainJS (взаимодействие с большими языковыми моделями)
- SemanticScholarJS (взаимодействие с Semantic Scholar)

Для начала работы с AcademiaOS вам требуется установленный [Node.js](https://nodejs.org/en/download).

1. Используйте `git clone` для клонирования этого репозитория.
2. Запустите `npm install`.

## 👨‍💻 Режим разработки

### `npm start`

Запускает приложение в режиме разработки.\
Откройте [http://localhost:3000](http://localhost:3000) для просмотра приложения в браузере.

Приложение автоматически перезагружается при внесении изменений.\
Ошибки линтинга отображаются в консоли.

### `npm test`

Запускает тестовый раннер в интерактивном режиме.\
Раздел о [запуске тестов](https://facebook.github.io/create-react-app/docs/running-tests) для дополнительной информации.

## 🏭 Продакшн-сборка

### `npm run build`

Компилирует приложение для продакшена в папку `build`.\
Эффективно собирает React в продакшн-режиме и оптимизирует сборку для максимальной производительности.

Бандл минифицирован, имена файлов содержат хеши.\
Ваше приложение готово к деплою!

Раздел о [деплое](https://facebook.github.io/create-react-app/docs/deployment) для подробностей.
